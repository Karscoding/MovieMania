Informatie over Movie Mania

Introductie

Movie Mania is een website gemaakt door twee studenten, Lucas Bruins en Kars Hekman.
De website is gemaakt als eindopdracht voor het vak Webtechnologie bij de opleiding HBO-ICT op de Hanzehogeschool in Groningen.

The Purpose

Het doel van Movie Mania is om informatie te beheren over films.
Met behulp van onze lijst kun je informatie vinden over films op een gemakkelijke manier.
Als laatst is het grootste nut van deze website natuurlijk wat wij hebben geleerd.
We hebben nieuwe dingen geleerd in Python, niet alleen de modules specifiek maar ook algemene Python functionaliteit.
Ook hebben we meer kunnen oefenen met HTML / CSS en zijn we over het algemeen is onze ICT kennis enorm vergroot.

De werking

De webpagina's zijn gemaakt met simpele HTML en CSS.
Wij hadden de optie om Bootstrap te gebruiken, dit zorgt ervoor dat je bijna geen CSS hoeft te schrijven maar wij vonden dat juist wel leuk.
Hierdoor hebben wij gekozen om geen Bootstrap te gebruiken en juist de uitdaging van CSS aan te gaan.
De werking van de website is gedaan met Flask in Python. Deze module zorgt voor veel functionaliteit.
De films in de lijst staan in een database, deze database is gemaakt met SQLAlchemy.
Ook de accounts staan in een database maar wel met een beschermd wachtwoord.

User Guide

Als gebruiker vraag je je misschien af hoe je de website gebruikt.
Als eerst moet je een account hebben en ingelogd zijn om bij de films te kunnen komen.
Registreren kun je doen via de Inlog-pagina.
Log hierna in en je kunt vervolgens bij de Film-lijst.
Wanneer je als gebruiker Admin bent kun je de Films aanpassen.

Film Toevoegen

Om een film toe te voegen moet je eerst ingelogd zijn als admin (lees volgend stuk).
Bovenaan de film lijst staat een knop on films toe te voegen.
Hiervan zijn de meeste dingen vanzelf sprekend behalve image.
Om hier een goed resultaat uit te krijgen moet je een film poster kiezen.
Hierbij moet je de image address kopieren en die invullen in het formulier van een film toevoegen.
